JS & jQuery Flashcards

Operators

An object that is capable of manipulating a value.

Variables

A location capable of storing temporary data.

Functions

A group of instructions used to return a single result or a set of results.

Methods

A function that is a member of a class or object.

Arguments

A value that is passed into a command, function, or routine.

Parameters

Refers to unknown data values capable of being set to a command set by the user.

Strings

A series of characters that are interpreted literally by a script.

Booleans

A value that is either true or false.

Attributes

Dictates certain aspects of an element.
